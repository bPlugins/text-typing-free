# Typing Text Block

Add animated typing effects to your headings and text directly in the WordPress Gutenberg block editor.

[Typing Text Block](https://bplugins.com/products/text-typing/) is a lightweight WordPress plugin that adds realistic animated typing effects to your headings and text — directly inside the Gutenberg block editor, with no coding required.

[![Typing Text Block Banner](https://ps.w.org/text-typing/assets/banner-772x250.png)](https://bplugins.com/products/text-typing/)

## 🚀 Key Features – Free Version
- **Animated Typing Text Block**: Add a clean, fully responsive typing animation anywhere in the block editor.
- **Prefix & Suffix Text**: Add static text before and after the animated typing lines.
- **Multiple Typing Lines**: Add unlimited rotating lines, each with its own color.
- **Typing Controls**: Configure Type Speed, Start Delay, Back Speed, Back Delay, Shuffle, Fade Out Effect, Loop, and Cursor.
- **Typography Controls**: Customize fonts for the prefix, typing, and suffix text.
- **Shortcode Generator**: Build a saved typing text and reuse it anywhere with a simple shortcode.
- **Optimized Performance**: Scripts load only when the block is used, keeping pages fast.

## 🛠️ Installation
1. Log in to your WordPress dashboard.
2. Go to **Plugins > Add New**.
3. Search for "Text Typing".
4. Click **Install Now** and then **Activate**.

## 📖 How to Use
1. Open the Block Editor (Gutenberg) on any Page or Post.
2. Search for the **Text Typing** block.
3. Add your prefix, typing lines, and suffix text in the block settings.
4. Customize speed, delay, looping, cursor, and styling options.
5. Publish your page and enjoy your animated typing effect!
